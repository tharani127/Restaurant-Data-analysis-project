# Restaurant-Data-analysis-project
This project involves analyzing a restaurant dataset to extract meaningful insights using Python, Pandas, and Matplotlib.
Task 1 Data Loading & Exploration
*Loaded the restaurant dataset using Pandas.
*Inspected dataset structure, columns, and basic information.
*Checked for missing values and understood data types.
*Performed initial exploratory data analysis to understand the dataset.
Task 2: Top Cuisines Analysis
*Identified the most commonly offered cuisines.
*Calculated the percentage share of top cuisines.
*Visualized top cuisines using bar charts.
*Derived insights on cuisine popularity.
Task 4: Price Range Distribution
*Analyzed the distribution of restaurants across different price ranges.
*Calculated the number and percentage of restaurants in each price range.
*Visualized price range distribution using bar plots.
*Interpreted affordability trends among restaurants.
